# Galaxy Robot Academy

Galaxy Robot Academy is a project-based software engineering academy for two young students, Parshan and Adryan, built around a continuous space robot story.

The goal is not to teach Python as a list of syntax rules. The goal is to teach how software engineers think: breaking problems down, debugging, experimenting, and building visible progress through small missions.

## Product Idea

Students are junior space software engineers at Galaxy Robot Academy. Their mission is to bring R0-B0, a space exploration robot, to life one capability at a time.

Every session unlocks a new robot skill:

- Speaking
- Remembering
- Asking questions
- Making decisions
- Repeating work
- Navigating
- Managing inventory
- Using reusable skills
- Exploring planets
- Surviving random events
- Preparing for launch
- Completing the final mission
