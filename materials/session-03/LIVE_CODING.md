## Personalized message


## Multiple questions

```python
name = input("Name: ")
planet = input("Favorite planet: ")
role = input("Mission role: ")

print(f"Engineer {name} will serve as {role}.")
print(f"Preparing a route to {planet}.")
```

## Numeric input

```python
age = int(input("Age: "))
print(f"Next year you will be {age + 1}.")
```

## Registration console

```python
name = input("Name: ")
```
