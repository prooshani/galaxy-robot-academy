# Galaxy Robot Academy
## Session 1 — Mission 01: Wake Up the Robot 🤖🚀

**Duration:** 75–90 minutes

## Mission Story

Welcome to **Galaxy Robot Academy**.

You are the newest software engineers of the academy.

Far away in the Andromeda Galaxy, an exploration robot named **R0-B0** has just been built.

The robot has powerful motors and sensors...

…but there is one problem.

**It doesn't know how to think yet.**

Only two young software engineers can teach it.

Today's mission:

> **Wake up R0-B0 and teach it to speak.**

---

# Learning Goals

By the end of this lesson students will:

- Understand what software is.
- Understand that computers only follow instructions.
- Write and run their first Python program.
- Use `print()`.
- Learn that programming is simply giving clear instructions.

---

# Teacher Guide

## 1. Ice Breaker (10 min)

Ask:

- What is software?
- What would happen if your Xbox had no software?
- Can a robot think by itself?
- What would YOU teach your own robot first?

There are no wrong answers.

---

## 2. Demonstration (10 min)

Tell them:

Imagine you tell your robot:

Go forward.

Turn left.

Say Hello.

If you forget one instruction...

The robot stops.

Programming is simply writing instructions one after another.

---

## 3. First Python Program (15 min)

Create:

```python
print("Hello Galaxy!")
```

Then

```python
print("Hello Parshan!")
print("Hello Adryan!")
print("Mission Started!")
```

Explain:

Python reads from top to bottom.

Every print is one instruction.

---

## 4. Experiment Time (15 min)

Challenge them to make the robot say:

- Good morning
- I am R0-B0
- I like pizza
- I want to explore Mars
- I found an alien!

Let them invent funny lines.

---

## 5. Mini Challenge (15 min)

Mission Card:

Create a robot introduction with at least six print statements.

Bonus XP:
- Make it funny.
- Add emojis.
- Tell a short story.

---

## 6. Debugging Game (10 min)

Show:

```python
print(Hello)
```

Ask:

Why doesn't it work?

Then:

```python
print("Hello")
```

Explain quotes.

---

# Quick Quiz

1. Does a computer think?
2. What does `print()` do?
3. Which runs first?
4. Why are quotation marks important?
