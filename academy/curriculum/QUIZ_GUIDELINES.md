# Quiz Guidelines

- 5–8 questions per session.
- 5–8 minutes.
- Unlimited retries.
- Passing threshold around 70%.
- GE awarded once.
- Immediate explanation after submission.
- Include output prediction, bug finding, concept understanding, and application.
- Avoid trick wording.
- Use code formatting for code.
- A failed attempt recommends review and retry; it does not remove GE.
