# Engineering Handbook

This folder contains the authoritative engineering documentation for Galaxy Robot Academy. Every engineer and AI contributor must read, understand, and adhere to these documents before making changes to the repository.

The handbook defines the project's engineering philosophy, delivery workflow, architecture principles, coding standards, task and prompt templates, review expectations, definition of done, and guidance for AI implementation agents.

When guidance conflicts or is unclear, follow the Engineering Constitution and request an architecture decision before proceeding.
