# Roadmap

## Phase 1: Foundation

Goal: make the academy coherent before building the platform.

Deliverables:

- Academy foundation
- Ranking and Galaxy Energy system
- Badge model
- Session template
- Homework mission template
- First two sessions

## Phase 2: Website PRD

Goal: define the student and teacher product before implementation.

Deliverables:

- Student journey
- Teacher journey
- Sitemap
- Data model
- Mission submission flow
- Badge and GE awarding flow
- Firebase schema
- Initial security rules direction
- UI and visual language

## Phase 3: Teaching Materials

Goal: produce complete class-ready material.

Deliverables:

- 12 session guides
- 12 homework missions
- Quick quizzes
- Debugging games
- Teacher notes
- Student mission sheets

## Phase 4: Platform MVP

Goal: build a usable website for the first cohort.

Deliverables:

- Next.js app
- Firebase Auth
- Firestore data model
- Student dashboard
- Teacher dashboard
- Mission listing
- Mission submission
- Manual teacher review
- GE and badges

## Phase 5: Launch

Goal: run the academy with real students.

Deliverables:

- Published Vercel site
- Seeded missions
- Student accounts
- Teacher review flow
- First session published
