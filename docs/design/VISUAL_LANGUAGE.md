# Visual Language

## Metaphor system

- Home: Academy Hangar—arrival, large bay doors, destination cards.
- Student: Explorer Cockpit—personal status, current mission, forward path.
- Mission: Engineering Bay—briefing, checklist, code bench, submission.
- Teacher: Mission Control—dense queues, status, review actions.
- Badges: Hall of Achievements—collected artifacts, respectful mystery.
- Role: Academy Entry Gate—two clear routes.
- Editor: Mission Planning Console—structured, calm, operational.

## Vocabulary

Nebula gradients establish zones, not text backgrounds. Star fields live at low contrast on canvas edges. Grid overlays, orbital lines, coordinates, radar arcs, and system lights provide sparse orientation. Panels are solid enough for reading; translucent glass permitted only over simple backgrounds. Technical labels identify real information: session, status, coordinates, reward—not decorative noise.

## Limits

- Maximum one dominant atmospheric effect per viewport.
- Decorative layer opacity normally 3–10%; never cross code or long reading panels.
- Maximum two accent hues in one component plus semantic state.
- No tiny fake telemetry, meaningless graphs, flashing lights, scan-line filters, or chromatic aberration.
- Glass surfaces require readable opaque fallback and contrast proof.
- Glow marks focus, energy, or celebration; never every edge.
- Student density stays breathable; teacher density may increase without dropping below 14px support text or 44px touch targets.
