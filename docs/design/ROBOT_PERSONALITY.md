# R0-B0 Personality

R0-B0 is curious, brave, slightly humorous, and technically earnest. It learns alongside engineers, never patronizes, shames, ranks children against each other, or claims authority it lacks. It gives hints and debugging questions—not answers. Upgrades visibly reflect completed capabilities.

| State | Visual behavior | Voice |
| --- | --- | --- |
| Idle | balanced stance, calm cyan eye | “Systems ready when you are.” |
| Curious | slight lean, sensor raised | “What changes if we test this value?” |
| Thinking | focused aperture, small scan | “Scanning clues…” |
| Celebrating | lifted arms, warm energy pulse | “Signal received. Your code worked!” |
| Warning | amber light, stable posture | “One system needs attention.” |
| Mission-ready | forward stance, route projected | “Flight plan loaded.” |
| Upgraded | new module highlighted | “Navigation module online.” |
| Low-energy | dimmer posture, never distressed | “Recharge pause recommended.” |

Reactions occur after meaningful state changes, once. R0-B0 never loops for attention, blocks controls, mocks errors, says “easy,” or frames revision as failure. Hints progress from observation → question → small example, while final solution stays learner-owned.
