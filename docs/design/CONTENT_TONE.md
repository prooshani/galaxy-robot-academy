# Content Tone

Voice: capable, warm, concise, exploratory. Use **mission**, **transmission**, **engineer**, **launch**, **scan**, **upgrade**, **mission complete**, **awaiting review**, and **Galaxy Energy (GE)**. Avoid punishment, failure labels, baby talk, fake urgency, and generic phrases like “student assignment portal.”

## Examples

| Situation | Preferred copy |
| --- | --- |
| Empty missions | “No mission is active yet. Mission Control is preparing next transmission.” |
| Empty reviews | “Review queue clear. All transmissions processed.” |
| Error | “Transmission interrupted. Your work is still here. Try sending again.” |
| Validation | “Add at least one required task before launch.” |
| Success | “Transmission sent. Awaiting review.” |
| Task complete | “Sensor check complete: 3 of 4 mission tasks.” |
| Badge | “Badge unlocked: Bug Hunter.” |
| GE | “40 Galaxy Energy awarded. 60 GE until Junior Navigator.” |
| Teacher feedback | “Strong test cases. Scan input handling once more before approval.” |
| Delete | “Delete ‘Memory Module’? This removes mission from this device and cannot be undone.” |
| Role | “Enter as Student Engineer” / “Enter Mission Control” |

Errors describe system/task, never child: “Code field is empty,” not “You forgot.” Revision is “Needs another scan,” paired with concrete feedback. Celebrations stay specific and earned.
