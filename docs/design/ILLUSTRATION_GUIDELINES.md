# Illustration Guidelines

Art direction: premium editorial space engineering—clean silhouettes, tactile machinery, soft nebula light, approachable scale. Not preschool clip art, hyperviolent military sci-fi, photoreal horror, or noisy game loot.

- R0-B0: compact modular explorer robot, expressive optical unit, visible upgrade ports, no human stereotype.
- Spacecraft/equipment: plausible controls and construction, simplified for ages 9–12.
- Planets/environments: scientific variety, strong focal point, safe wonder.
- Mission patches/badges: bold silhouette, limited palette, recognizable at 48px; secret badges retain mystery.
- Diagrams: instructional first; label real relationships; no decorative pseudo-schematics.
- Expressions: readable through pose/light/eye shape, never infantile.

Export master vector where possible; UI SVG for icons/patches, WebP/AVIF plus PNG fallback for raster, transparent background for characters/equipment. Provide 1×/2× raster sizes, descriptive filenames, alt guidance, license/source metadata. Define focal safe zone; responsive crops never remove R0-B0 face, action, or instructional labels. Background art may be `aria-hidden`; meaningful diagrams need text equivalent.
