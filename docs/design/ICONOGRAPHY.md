# Iconography

Use one rounded outline family with 1.75–2px stroke, simple geometry, and engineering clarity. PRD recommends Lucide, but audit confirms it is **not installed**; adoption requires separate dependency approval. Until then, use coherent project SVGs or text without pretending library availability.

Sizes: 16px inline, 20px control, 24px navigation/card, 32px feature. Filled form indicates selected/earned only when matching outline exists. Semantic icons keep stable meaning: check complete, clock reviewing, lock locked, warning triangle attention, bolt/energy GE. Icon-only controls require accessible names and 44px targets; unfamiliar icons retain visible labels.

Emoji acceptable in informal content or temporary mascot expression when marked decorative and platform variation is harmless. Never use emoji for core navigation, status, destructive actions, badges requiring consistent art, or technical diagrams. Do not mix multiple icon families, line weights, 3D stickers, and emoji in one control system.
